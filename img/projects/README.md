# Project Images

Place your project images here with the following names:

- `wiki-refactor.jpg` - AWS Service documentation wiki refactor
- `ai-upskilling.jpg` - AWS Service documentation AI upskilling program  
- `task-migration.jpg` - AWS task management tool migration
- `multi-company.jpg` - Multi-company documentation integration

## How to download images from Notion:

1. Open your Notion page: https://www.notion.so/2a34a714c86580a5adc7da8ffe0a112b
2. Right-click on each project image
3. Select "Save image as..." or "Download image"
4. Save them with the names listed above in this `img/projects/` folder

Alternatively, you can:
- Export the entire Notion page as HTML/images
- Use browser developer tools to find image URLs
- Take screenshots of the project sections
